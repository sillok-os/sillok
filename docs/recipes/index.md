# Recipes

Short, single-purpose snippets. Each recipe is ≤ 1 screen and copy-pasteable.

> Tutorials walk you through a full scenario end-to-end. Recipes solve
> one concrete problem fast.

## Conventions

Each recipe is a single H2 in this file (or, when long, a separate `.md`
file linked from this index). Recipe titles are imperative ("Print …",
"Replay …", "Force-trigger …").
